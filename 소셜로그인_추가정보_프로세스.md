# 네이버·카카오 소셜 회원가입 후 추가정보 수집 프로세스

## 1. 공통 전제

- 소셜 로그인은 최소 정보만 수집
- 추가정보는 서비스 가입 단계에서 별도 수집
- 필수: 이용약관, 개인정보 수집·이용 동의

## 2. 네이버 회원가입 프로세스

1. 네이버 로그인 (OAuth)
2. naver_id 수신 및 회원 식별
3. 추가정보 입력 페이지 이동
4. 동의 체크
5. 회원가입 완료

## 3. 카카오 회원가입 프로세스

1. 카카오 로그인 (OAuth)
2. kakao_user_id 수신 및 회원 식별
3. 추가정보 입력 페이지 이동
4. 동의 처리
5. 가입 완료

## 4. UX 권장 흐름

    소셜 로그인
     → 최소 계정 생성
     → 추가정보 입력 요청
     → 정회원 전환

## 5. 실무 체크리스트

- 소셜 고유 ID로 회원 매핑
- 추가정보 자체 DB 관리
- 필수/선택 항목 명확화
- 약관 버전 관리
- 카카오 심사용 화면 준비

## 6. 요약

소셜 로그인 후 첫 로그인 시 추가정보를 수집하여 정회원으로 전환한다.
