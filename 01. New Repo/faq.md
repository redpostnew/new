### 📋 사방넷/배송대행 FAQ 총정리

| 분류 | 질문 | 답변 |
| :--- | :--- | :--- |
| **📦 1. 배송 및 운송** | **배송 기간은 얼마나 걸리나요?** | • LCL/FCL: 출항 후 약 2일 소요 (인천 기준)<br>• 해운특송: 평균 5~7일 소요<br>• 항공: 4~5일 소요<br>※ 기상 및 세관 통관 상황에 따라 변동될 수 있습니다. |
| | **합배송이 가능한가요?** | 네, 가능합니다. 단, 구매대행 화물과 배송대행 화물은 섞어서 합배송할 수 없습니다. |
| | **출고 후 배송 조회는 어떻게 하나요?** | • 해운특송/항공: 출고 완료 후 송장번호 안내 (한국 택배사 홈페이지 조회)<br>• FCL/LCL: B/L 번호 안내 (유니패스에서 조회 가능) |
| | **중국 내 배송비는 어떻게 되나요?** | 판매처에서 위해(Weihai) 창고까지 이동하는 배송비는 상품 주문 시 발생합니다. (판매자 정책에 따라 무료 또는 유료) |
| | **출고 마감 시간은 언제인가요?** | 평일 오후 4시 마감입니다. 이후 결제 건은 다음 날(익일) 출고됩니다. |
| | **무게를 줄일 수 있나요?** | 네, 박스 제거, 과대 포장 제거, 합포장 등을 통해 가능합니다. 단, 파손 위험이 증가할 수 있습니다. |
| | **배송 중 분실되면 어떻게 되나요?** | 운송사 확인 후 보상 절차가 진행됩니다. 운송 규정 한도 내에서 보상되며 처리 시간이 다소 소요될 수 있습니다. |
| | **위해에서 직접 픽업도 가능한가요?** | 네, 가능합니다. 단, 사전 예약이 필수이며 픽업 관련 자료를 제출해 주셔야 합니다. |
| | **대형 화물(팔레트)도 가능한가요?** | 네, 팔레트 작업이 가능합니다. LCL/FCL 해상 운송으로 진행되며 주로 사업자 전용이나 쿠팡 입고용으로 사용됩니다. |
| | **LCL 최소 CBM 기준이 있나요?** | 0.1 CBM부터 진행 가능하지만, 안전을 위해 최소 0.5 CBM 이상을 권장합니다. |
| | **FCL과 LCL의 차이는 무엇인가요?** | • LCL: 여러 업체의 화물을 하나의 컨테이너에 같이 싣는 방식(혼적)<br>• FCL: 한 업체가 컨테이너 한 대를 단독으로 사용하는 방식 |
| **🏛️ 2. 통관 및 세무** | **개인통관고유부호가 꼭 필요한가요?** | 네, 개인 수입 통관 시 필수입니다. 수취인 명의와 통관번호가 일치해야 하며, 통관부호 신청 주소지와 수취지가 동일해야 합니다. |
| | **관세와 부가세는 언제 발생하나요?** | • 개인 통관: 미화 150달러 초과 시 발생<br>• 사업자 통관: 금액 무관하게 무조건 발생 |
| | **금지 품목을 보내면 어떻게 되나요?** | 세관에서 폐기 또는 반송 처리되며, 이때 발생하는 비용은 고객 부담입니다. |
| | **KC 인증이 필요한 상품도 배송 가능한가요?** | 자가사용 목적은 일부 가능하지만, 판매 목적(사업자 통관)은 KC 인증이 필수입니다. 특히 전자제품/전기용품은 사전 문의가 필수입니다. |
| | **세금계산서 발행 가능한가요?** | 네, 가능합니다. 상세한 발행 항목은 별도로 문의해 주셔야 합니다. |
| | **통관 보류가 되면 어떻게 되나요?** | 세관에서 구매 내역서, 결제 내역 등의 추가 자료 제출을 요청합니다. 제출 지연 시 창고 보관료가 발생할 수 있습니다. |
| | **세관 검사는 얼마나 자주 발생하나요?** | 기본적으로 무작위(랜덤) 선정입니다. 단, 신규 사업자, 고위험 품목, 반복 수입, 신고 금액 편차가 클 경우 검사 확률이 높아집니다. |
| | **관세 및 부가세는 언제 납부하나요?** | 수입 신고 후 세금 고지서가 발행되며, 납부 완료 후 화물 반출이 가능합니다. (관세사 대납 기능 지원) |
| | **원산지 증명서(C/O) 발급도 가능한가요?** | 네, 가능합니다. 한중 FTA 적용 가능 품목에 한해 관세 절감이 가능하며 사전 협의가 필수입니다. |
| | **위험물도 취급 가능한가요?** | 매우 제한적으로 가능합니다. UN 번호, MSDS(물질안전보건자료) 제출이 필수이며 사전 승인 없이는 선적이 불가합니다. |
| | **단가를 낮추기 위해 신고 금액을 조정할 수 있나요?** | 절대 불가합니다. 수입 신고는 실제 거래가 기준이며, 허위 신고(언더밸류) 적발 시 세금 추징, 과태료 발생 및 향후 통관에 불이익을 받습니다. |
| | **한중 FTA 적용 시 관세가 얼마나 절감되나요?** | 수입하시는 개별 품목(HS Code)별로 절감률이 모두 상이합니다. |
| **🏭 3. 창고 및 입출고** | **검수는 어떻게 진행되나요?** | • 구매대행: 기본 검수 (색상/사이즈/수량 전수확인, 불량/오염 랜덤 5개 이하)<br>• 배송대행: 송장번호 창고 입고 여부 및 원산지 표기 확인 |
| | **보관 기간은 얼마나 되나요?** | 창고 입고 후 기본적으로 제공되는 무료 보관 기간은 30일입니다. |
| | **상품이 일부만 도착했어요. 어떻게 되나요?** | 먼저 도착한 상품은 시스템에 '부분 입고'로 등록되며, 모든 상품 도착 후 합배송을 요청하실 수 있습니다. |
| | **재포장도 가능한가요?** | 네, 가능합니다. 박스 제거 후 재포장, 에어캡 보강, OPP 개별 포장, 브랜드 로고 제거 등이 지원됩니다. |
| | **반품도 가능한가요?** | 한국 도착 후에는 반품이 불가합니다. 중국 창고 입고 후 출고 전 상태에서 실사 사진 확인 후 요청하시면 반품 가능합니다. |
| | **상품 사진 촬영 서비스가 있나요?** | 네, 상품 입고 시 확인을 위한 기본 사진 촬영은 무료로 제공됩니다. |
| | **폐기 요청도 가능한가요?** | 네, 불량 상품이나 통관 불가 상품에 대해 폐기 처리를 요청하실 수 있습니다. |
| **🤝 4. 부가 서비스** | **OEM / 라벨 작업도 가능한가요?** | 네, 스티커 부착, 라벨 교체, 단순 조립, 합포장 등이 가능합니다. (대량 작업은 사전 상담 필요) |
| | **중국 공장 직거래도 연결해주시나요?** | 네, 공장 소싱, 샘플 구매, 단가 협상 지원, 검품 후 발송까지 모두 지원 가능합니다. |
| | **상품 가격 협상도 해주시나요?** | 당사의 '구매대행' 서비스를 이용하시는 고객에 한하여 1688 등 플랫폼의 할인 가격 적용을 지원해 드립니다. |
| **📞 5. 고객 서비스** | **고객센터 운영 시간은 어떻게 되나요?** | 평일 09:00 ~ 18:00 운영 (주말 및 공휴일 휴무). 긴급 출고 건은 사전 협의가 필요합니다. |
