#  👑 주문번호(코드) 체계

## 1. 설명

- 주문번호 체계는 주문을 유일하게 식별하기 위한 규칙
- 쇼핑몰·물류·회계·API 등 시스템별로 다르지만 기본 구조는 거의 같음
- 날짜 + 구분코드 + 일련번호 + 고객번호

주문번호 체계는 주문을 유일하게 식별하기 위한 규칙입니다.

| 구성 요소 | 예시              | 설명                 |
| -----    | --------------- | ------------------ |
| 날짜 코드 | 251017        | 주문일자(YYMMDD)     |
| 구분 코드 | B / G / S| 주문 출처(배송/구매/수입) |
| 일련번호  | 001          | 하루 단위 순번 3자리 |
| 고객번호  | 0099          | 고객번호 4자리 |
| 합친 형태 | 250117-B123-0099 | 하나의 주문번호 완성        |
