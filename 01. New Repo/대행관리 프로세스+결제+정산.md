# 대행관리 프로세스&결제&정산

## 배송대행 
### 개요
- 1688 기반으로 상품 직접 결제 기능없음
- 결제는 고객이 1688 실제 판매자 페이지로 이동해서 진행
- 고객의 결제 프로세스는 외부(1688)에서 처리
- 레드포스트는 단순히 [상품찾기]를 통해 장바구니와 결제지원만 하며, 결제와 통관은 고객이 직접 진행함
- 해외 사용자는 결제 제한이 많기 때문에 대행업체(소싱대행사)를 통해 결제하는 경우가 일반적
### 결제 방식

- 알리페이(Alipay, 支付宝) 사용이 기본
- 중국 내 은행계좌나 카드필요 필요

### 배송대행 절차
#### 슈퍼레이블 참조 / https://super-label.com/
#### 주문/검수
    0. 장바구니: 고객이 검색 후 장바구니 담기
    1. 결제대기: 장바구니에 담고 견적신청한 상태, 주문번호 생성
          1. 상품을 선택하고 1688 [주문서 생성]
          2. [주문서 생성]을 하면 1688 본인 계정에도 주문서가 들어감
          3. 결제전 고객이 판매자와 가격협상 및 소통하여 단가 수정
          4. 신용카드/카카오페이/알리페이/월드퍼스트(1688에 결제)
             - 1688 연동(처음 1회만 필요)
             - 1688 직접결제>알리페이(쯔프바오)결제
             - 1688 직접결제>카드등록 및 카드결제
             - 1688 직접결제>카카오페이 결제
             - 월드퍼스트 결제
                - 월드퍼스트 계정 연동(처음 1회만 필요)
    2. 구매완료
       1. 결제완료하면 구매리스트에 노출
       2. 체크박스로 클릭하여 배송대행 신청서 작성
       3. or 엑셀로 작성후 신청
    3. 접수신청
    4. 입고완료
    5. 재고출고요청: 
       - 창고에 있는재고를 출고요청(창고->출고전 상품임)
       - 고객요청이 있을 때에만 노출함
    6. 검수완료
    7. 출고요청
    8. 포장완료
    9. 결제대기
#### 배송
    1.  결제완료: 무통장입금/신용카드/예치금 = 레드포스트에 결제
    2.  출고완료 
    3.  배송완료

### 결제 방법
- 무통장 입금
- 신용카드/체크카드 - 나이스페이먼츠
  - 신용카드 수수료 적용
- 실시간 이체
- 가상계좌
- 예치금
  - 원화 입금계좌
  - 외화 입금계좌(자동 환율 계산 및 결제 수수료 표시 기능)

### 특징
- 결제 이후 단계(수입신고·배송대행)까지 시스템에서 관리 가능
- 구매-결제-배송 과정을 통합 관리 가능
- 1688 결제 경험이 없는 사용자도 원화로 쉽게 결제할 수 있게 설계함
- 관리자는 실시간 마진 계산, 정산 리포트
