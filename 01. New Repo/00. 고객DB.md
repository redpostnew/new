# 고객 DB 구조

## 1. 기본정보
- **사서함**
- **이름**
- **휴대전화번호**
- **이메일**
- **가입경로**
- **배송지** (=배송지 DB 연동)
- **추천인**
- **회원등급**
    - `일반회원`: 기본 가입 시 부여
    - `실버회원`: 최근 6개월 구매금액 1,000만원 이상
    - `골드회원`: 최근 6개월 구매금액 3,000만원 이상
    - `VIP회원`: 최근 6개월 구매금액 7,000만원 이상

## 2. 추가정보
- **가입경로**
- **회원분류**
    - 개인회원
    - 사업자회원 (사업자등록증)
    - 관리자
- **머니정보**
    - KRW 예치금
    - CNY 예치금 (고객이 신청 시 환율조회 값 -> 관리자가 확인 후 예치함)
    - 쿠폰 (미래)
