# 상품분류 = 카테고리 = 품목
## 원칙 (1688 기준)
- 1688 카테고리 기준으로 상품분류
  - 1차 대분류는 총 11개
  - 2차 중분류는 총 63개
  - 3차 소분류는 총 419개
  - 위 분류는 실제 검색 항목으로 쓰인다고 함
  - <img src="https://i.ibb.co/kszXqSvt/Snipaste-2025-11-18-14-33-16.png" alt="Snipaste-2025-11-18-14-33-16" border="0">
- 1688 원래 카테고리는 사용자화면에 쓰기 어려움
  - 1차 대분류는 58개
  - 2차 중분류는 1,468개
  - 3차 소분류는 11,418개
  - 분류와 용어들이 고객용보다는 내부 관리용으로 보임
- 활용
  - 상품번호에도 적용함
  - hs_code는 해당 카테고리일 경우에 붙임
  - hs_code는 소재별로 구분하기 때문에 상품 카테고리에는 중복값이 있음
  - hs_code는 CLO10-12 셔츠의 HS Code가 6206.30 (면/인조섬유 직물)이지만, 만약 실크 셔츠라면 다른 HS Code(예: 6206.10, 6206.90 등)이 다름
  - hs_code는 해당 3차 분류 품목에서 가장 일반적인 대표 HS Code로 사용해야 함
- 예외
  - 혹시 상품이 아래 카테고리 분류에 들어오지 않을 경우 (논의 필요)
