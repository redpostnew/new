# 출고 단계 (중국 → 국내 발송)

## 1. 출고 정보
### **1. 개요**
- 출고 전 라벨에 QR을 인쇄하여 고객 주문정보, 통관정보, 운송장 번호를 확인
- 스캔만으로 출고 기록을 자동 체크합니다.

### 2. 주요 기능
* **QR 시스템 적용**: 라벨 인쇄 시 QR 코드 포함
* **자동화**: 스캔만으로 출고 기록 자동 체크

### 3. 주문 정보 (주문 DB)
* **신청 방법**: 배송대행 / 구매대행 / 수입대행
* **수입 방식**: 개인전자상거래 / 병행수입
* **통관 정보**: 고객번호 (사서함번호)
* **수령인 정보**: 받는 사람 (고객 DB 주소록 연동)
* **기타**: 고객 요청사항, 결제완료 여부, 검수완료 여부

### 4. 도입 효과
* **오배송 방지**
* **고객 편의성**: 고객이 QR 스캔 시 배송 추적 가능
* **업무 효율**: 출고 처리 속도 향상

## 2. 진행 순서

### 1단계: 나눔/묶음 관리
1.  **구분** 
     - 나눔 
     - 묶음
2.  **상품수**
3.  **고객번호**
4.  **신청일**
5.  **주문번호** (주문 DB 연동)
6.  **합배번호 / 나눔번호**
7.  **송장번호**
8.  **배송상태**
    * 접수신청
    * 부분입고
    * 전체입고
9.  **출고지 지정**

### 2단계: 상품 및 주문정보 확인
1. 주문 정보 (주문 DB 연동)
   * **신청 방법**: 배송대행 / 구매대행 / 수입대행
   * **수입 방식**: 개인전자상거래 / 병행수입
   * **받는 사람**: 고객 DB 주소록 대조
   * **주요 확인 사항**:
       * 고객 요청사항
       * 결제완료 여부 확인

2. 검수 및 촬영 정보
   * 검수 작업 완료 여부 확인
   * 촬영(실사) 완료 여부 확인

### 3단계: 포장/패킹 관리
#### 1. 출고 정보 설정
* **출고 방법**
    * 재고등록 (중국창고): 포장하지 않음
    * 3PL 출고 (한국창고)
    * 쿠팡 바로출고 (쿠팡창고): 입고센터 확인 필요
    * 주소록 출고 (고객 지정 주소)
* **물류 방식**
    * 항공 / 해운택배 / 인천 LCL / 평택 LCL
* **신청 방법**: 배송대행 / 구매대행 / 수입대행
* **상품 정보**: 상품 DB 연동

#### 2. 추가 작업
* 라벨링 (바코드, 스티커 부착)
* 리패킹 (기존 포장 제거 후 새 박스 포장)

#### 3. 패킹 관리 (중국 → 한국 발송 시)
* **주문번호**: 주문 DB 연동
* **상품정보 (상품 DB)**: 상품명, 상품코드, 상품수량
* **상자정보**: 상자번호(카톤넘버), 색상
* **규격 및 중량**:
    * 부피(CBM): 가로 × 세로 × 높이
    * 무게(KG)
* **현장 처리**:
    * 포장명세서 작성
    * **패킹 엑셀 업로드**:
        * 일자 및 시간
        * 중량, 가로, 세로, 높이, 부피
        * 이름 및 상호
        * 부가서비스 코드 (58: 씨오, 59: 원산지작업, 60: 한국내 배송비)
        * 부가서비스 비용
* **포장 완료**

#### 4. 쿠팡 부착 문서 (쿠팡 출고 시)
1.  고객이 패킹관리 리스트 확인 후 쿠팡에 요청
2.  쿠팡에서 받은 문서를 시스템(레포)에 업로드
    * 물류 동봉 문서
    * 스토어 팔레트 정보

#### 5. 후속 절차
* 인보이스 발행
* B/L 생성 (LCL인 경우만 해당)
* 포장 확인 및 출고 검수
* 업무 설명

### 4단계: 출고 관리
#### 1. 출고 정보 확인
* 출고 시간 / 담당자
* 대행 종류: 배송대행 / 구매대행 / 수입대행

#### 2. 출고 상태 값
1.  출고대기
2.  고객 결제대기/완료
3.  포장완료
4.  출고검수 완료
5.  (운송사 인계완료)
6.  출고완료
7.  배송중
8.  배송완료

## 3. 출고 → 집하

### 1. 서류 준비
* **상업송장 (Commercial Invoice)**: 실제 거래 금액, 수량, 상품명 (세관 신고 및 결제 청구용)
* **패킹리스트 (Packing List)**
    * Numbers (품번/순번)
    * BOX Q’TY (박스 수량)
    * Q’TY (상품 수량)
    * CBM / CLR / SZ / KG (부피/색상/사이즈/무게)

### 2. 라벨 부착 및 CS 관리
* **라벨링**: 운송장 출력 후 부착 (운송사 정보 포함)
* **QR코드 활용 CS/사후관리**:
    * 상품 QR 스캔 시 고객센터 연결 및 주문정보 로딩
    * 최종 패키지 QR 부착 → 고객 스캔 시 [배송조회, 고객센터, 리뷰작성, 재구매 쿠폰] 제공
* **기대 효과**:
    * 고객 대응 속도 증가 및 CS 문의 감소
    * 주문정보 입력 오류 감소 및 소형상품 클레임 최소화
    * 고객 접점 확대 및 재구매 유도

### 3. 집하 및 선적
* 창고 → 항구/공항 이동
* 운송장 (B/L 또는 AWB) 발행 (LCL인 경우 B/L 등록 필수)

### 4. 운송 및 추적
* **운송사 인계**: 택배사 인수증 스캔 또는 바코드 정보 수신 시 '출고완료' 처리
* **운송 방법별 송장 생성**:
    * **해운**: B/L 발행 (운송장번호를 B/L로 대체, 포워더: Yida)
    * **항공**: 주문 시 생성 (Yida 연계, CJ대한통운)
    * **택배**: 주문 시 생성 (Yida 연계, CJ대한통운)
* **참고 사항**: 고객 발송 안내 방식 검토 필요
