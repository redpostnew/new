# 배송대행/구매대행 프로세스
## 배송대행 
### 직접 결제 기능 없음(1688 기반)
- 레드포스트는 1688 상품 정보를 ‘조회’만 제공
- 결제는 고객이 1688 실제 판매자 페이지로 이동해서 진행
### 결제 방식
- https://aibuy.1688.com/landingpage?bizType=purchasingAgent&customerId=redpost
알리페이(Alipay, 支付宝) 사용이 기본이며, 중국 내 은행계좌 필요.

### 결제 방법
- 신용카드
- 카카오페이
- 알리페이
- 월드퍼스트

### 특징
- 고객의 결제 프로세스는 외부(1688)에서 처리
- 레드포스트는 단순히 [상품찾기] 만 지원하며, 겨제와 통관은 고객이 직접 진행함
해외 사용자는 결제 제한이 많기 때문에 **대행업체(소싱대행사)**를 통해 결제하는 경우가 일반적입니다.

## 구매대행
### 어시스턴트 결제 구조 (통합 운영형)
#### 주소

-  내부 결제 연동 지원 가능
→ 소싱홈에서 찾은 상품을 실제 구매까지 연결하도록 설계된 확장 기능.
→ 결제대행·정산관리·수수료 계산 등 운영 단계의 지원이 포함됩니다.

### 결제 방법
- 무통장 입금
  - 원화 입금계좌
  - 외화 입금계좌(자동 환율 계산 및 결제 수수료 표시 기능)
- 신용카드/체크카드 - 나이스페이먼츠
  - 신용카드 수수료 적용
- 실시간 이체
- 가상계좌
- 예치금
  - 원화 입금계좌
  - 외화 입금계좌(자동 환율 계산 및 결제 수수료 표시 기능)
### 특징
- 결제 이후 단계(수입신고·배송대행)까지 시스템에서 관리 가능
- 구매-결제-배송 과정을 통합 관리 가능
- 1688 결제 경험이 없는 사용자도 원화로 쉽게 결제할 수 있게 설계험
- 관리자는 실시간 마진 계산, 정산 리포트



| 구분    | 소싱홈            | 어시스턴트              |
| ----- | -------------- | ------------------ |
| 결제 위치 | 외부(1688·대행사)   | 내부(아이템스카우트 연동)     |
| 결제수단  | 알리페이·위안화       | 원화·국제결제            |
| 목적    | 상품 찾기·조사       | 상품 구매·정산 관리        |
| 추가 지원 | 없음 (사용자 직접 처리) | 환율·수수료 자동계산, 통합 관리 |
| 적합 대상 | 직접 수입 셀러       | 국내판매·자동정산 원하는 셀러   |
