## 1. 화면 개요

- **페이지명** — 출고 관리
- **목적** — 전체 출고 프로세스 현황을 모니터링하고 배송 정보를 관리
- **주요 수행 업무**
  - 출고 대상 주문 조회 및 현황 모니터링
  - 패킹 처리 (박스 구성·치수 입력·상품 투입)
  - 통관 유형 선택 (일반통관·목록통관)
  - 쉬핑마크 출력 (박스별·팔레트별)
  - 쿠팡 서류 출력 및 관리
  - 출고 상태 변경 및 한국 운송장 출력

## 2. 출고 업무 흐름

### 2.1 정상 출고 흐름

```
출고요청
    ↓
패킹하기 (박스 구성·치수 입력·상품 투입)
    ↓
쉬핑마크 출력
    ↓
포장완료
    ↓
결제 처리
    ├─ 예치금·카드: 자동 처리
    └─ 무통장: API 입금 확인 후 이동
    ↓
결제완료
    ↓
출고완료 (결제 완료건만 확인하여 출고)
    ↓
배송완료 (주문서 최종 완료)
```

### 2.2 예외 흐름

```
포장오류 → 오류 유형 선택·메모 저장 → 재처리
출고보류 → 보류 사유 확인 → 해제 후 진행
쿠팡서류 대기 → 쿠팡 서류 출력 후 처리 → 서류대기 해소
```

### 2.3 상태 코드

| 상태 | 설명 |
|------|------|
| 출고요청 | 고객·관리자가 출고를 요청한 상태 |
| 포장완료 | 패킹 완료 후 상태 변경 |
| 포장오류 | 패킹 과정에서 오류 발생 |
| 쿠팡서류 대기 | 쿠팡 배송 서류 출력 대기 중 |
| 출고보류 | 출고 보류 처리된 상태 |
| 결제대기 | 비용 입력 후 고객 결제 대기 |
| 결제완료 | 결제 확인 완료 |
| 출고완료 | 출고 완료 처리 |
| 배송완료 | 최종 배송 완료 |

## 6. 패킹 처리

### 6.1 패킹 관리 모달 열기

- **방법 1** — 대상 주문 체크박스 선택 → [패킹하기] 버튼 클릭
- **방법 2** — [추가기능 ▾] → [패킹 내역 상세] 선택

### 6.2 패킹 관리 모달 구성

패킹 관리 모달은 **4개 섹션**으로 구성된다.

```
📋 주문 정보  |  📦 패킹 요약  |  🗂️ 박스별 상세 내역  |  🏗️ 팔레트 구성
```

### 6.3 박스 생성

1. [박스 생성] 버튼 클릭
2. 생성할 박스 수 입력
3. 박스번호 자동 부여 (예: BOX-001, BOX-002...)
4. 박스 목록에 표시 (박스번호 / 담긴 상품 수)

### 6.4 상품 투입 (박스에 상품 담기)

**방법 1 — 리스트에서 수량 입력**

1. 왼쪽 패널에서 박스 선택
2. 상품 전체 리스트에서 각 상품의 투입수량 입력
3. [적용하기] 클릭

**방법 2 — 바코드 스캔**

1. 박스 선택
2. 바코드를 스캔하면 해당 상품 자동 투입
3. 스캔 중 상태 표시 → 완료 후 투입 결과 표시

**방법 3 — 모바일**

- 모바일 기기에서 [박스선택] → 상품 바코드 스캔 → 자동 투입

### 6.5 박스 치수 입력

각 박스의 실측값을 입력한다.

| 항목 | 설명 | 단위 |
|------|------|------|
| 가로 | 박스 가로 길이 | cm |
| 세로 | 박스 세로 길이 | cm |
| 높이 | 박스 높이 | cm |
| 중량 | 박스 실제 무게 | kg |
| 부피무게 | 가로×세로×높이 ÷ 부피무게 계산 기준 | cbm |

- [건너뛰기] — 치수 입력 없이 다음 단계 진행

### 6.6 진행 현황 확인

| 항목 | 설명 |
|------|------|
| 전체 상품 | 투입 대기 중인 전체 상품 수 |
| 미투입 | 아직 박스에 담기지 않은 상품 |
| 패킹 완료 | 모든 상품 투입 완료 여부 |
| 팔레트 적재 대기중 | 팔레트 구성 대기 상태 |

### 6.7 패킹 요약 정보

| 항목 | 설명 |
|------|------|
| 주문유형 | 구매대행·수입대행·배송대행 |
| 물류방법 | 운송 방식 |
| 출고방식 | 출고 방법 |
| 총 박스 수 | 전체 박스 개수 |
| 팔레트 수 | 전체 팔레트 개수 |
| 총 CBM | 전체 부피 합계 |
| 총 중량 | 전체 중량 합계 |

## 8. 쉬핑마크 출력

### 8.1 박스별 쉬핑마크 출력

**출력 경로**

- 패킹 관리 모달 → [쉬핑마크 (상자)] 미리보기 탭
- 박스 선택 시 미리보기 표시

**출력 절차**

1. 패킹 관리 모달에서 출력할 박스 선택
2. 쉬핑마크 미리보기 확인
3. [쉬핑마크 출력] 버튼 클릭
4. 프린터 출력

**쉬핑마크 포함 정보**

| 항목 | 예시 |
|------|------|
| 받는사람 | 김 민 준 |
| 배송지 | 경기도 파주시 숲속노을로 9-15 |
| 보내는곳 | REDPOST 인천센터 |
| 박스 번호 | BOX-001 of 3 |
| 중량/치수 | 중량(kg) · 부피무게(cbm) |

### 8.2 팔레트 쉬핑마크 출력

**팔레트 구성 절차**

1. 패킹 관리 모달 → [팔레트 구성] 탭 선택
2. [상자를 선택해 팔레트에 적재해주세요] 안내에 따라 박스 선택
3. 팔레트에 박스 적재 구성 완료
4. [쉬핑마크 (팔레트)] 미리보기 확인
5. [쉬핑마크 출력 (팔레트)] 버튼 클릭

> 📌 팔레트 등록 후 쉬핑마크 미리보기 표시

### 8.3 박스 내부 부착용 출력

- 패킹 관리 모달 → [출력 미리보기] 탭
- 박스 내부 부착용 바코드·상품 정보 출력
- [상품 바코드 출력] 버튼 클릭

## 9. 쿠팡 서류 출력

### 9.1 쿠팡 서류 개요

- 쿠팡 납품을 위한 배송 서류를 출력한다.
- 쿠팡서류 대기 상태의 주문은 서류 출력 후 처리해야 다음 단계로 진행 가능

### 9.2 쿠팡 서류 출력 절차

1. 목록 테이블에서 쿠팡서류 열 링크 클릭
   또는 [추가기능 ▾] → [쿠팡서류] 선택
2. 쿠팡 배송 서류 모달 열림
3. 서류 내용 확인

  | 항목 | 설명 |
  |------|------|
  | 송장번호 | 배송 송장번호 |
  | 수취인 | 받는 사람 이름 |
  | 배송지 | 수취인 주소 |
  | 상품 명세서 | 출고 상품 목록·수량 |
  | 배송정보 | 배송 방식·센터 정보 |

4. [쿠팡서류 인쇄] 버튼 클릭

### 9.3 서류 오류 처리

서류에 오류가 있는 경우 오류 처리 절차 진행

1. [서류오류 처리] 버튼 클릭
2. 오류 유형 선택
3. 오류 내용 메모 입력
4. [저장] 클릭

### 9.4 쿠팡 서류 출고 센터 안내

|------|------|

## 10. 출고 상태 변경

### 10.1 일괄 상태 변경

1. 대상 주문 체크박스 선택
2. [상태변경] 드롭다운에서 변경 상태 선택
3. [변경] 클릭 → 확인 팝업에서 확인

### 10.2 변경 가능 상태

| 상태 | 설명 |
|------|------|
| 검수완료 | 검수 완료 처리 |
| 출고요청 | 출고 요청 처리 |
| 포장완료 | 패킹 완료 처리 |
| 포장오류 | 패킹 오류 처리 → 오류 사유 선택 필수 |
| 출고보류 | 출고 보류 처리 |
| 출고완료 | 출고 완료 처리 |
| 배송완료 | 최종 배송 완료 처리 |

### 10.3 포장오류 처리

- [포장오류] 상태 변경 시 **포장오류 사유 선택** 모달 표시
- 사유 선택 → 저장 시 이력에 기록
- **오류 사유 종류**
  - 상품 수량 불일치
  - 상품 파손
  - 오배송
  - 기타 (직접 입력)

### 10.4 출고 이력 확인

- 각 주문의 출고 처리 이력 조회 가능
- 담당자·처리 일시·이전 상태·변경 상태 기록

### 10.5 한국 운송장 처리

- **운송장 출력** — 한국운송장 열 프린터 아이콘 클릭
- **송장 재발급**
  1. [추가기능 ▾] → [송장 재발급] 선택
  2. 기존 송장 확인
  3. [재발급 송장] 생성
  4. 재발급 송장 출력

## 11. 나눔(Split) · 묶음(Merge)

### 11.1 나눔 처리

- **목적** — 상품을 사이즈·종류별로 나눠 배송
- **활성화** — 체크박스 2건 이상 선택 시 [나눔] 버튼 활성
- **절차**
  1. 대상 주문 체크박스 선택 (2건 이상)
  2. [나눔] 클릭
  3. 현재 주문정보 및 이미지·배송센터·배송구분 확인
  4. 나눌 제품 목록에서 상품·수량 설정
  5. [나눔 확정] 클릭
- **나눔 처리 내역** — 원주문번호 / 나눔수량 / 잔여수량 표시
- ⚠️ 처리 후 새 주문번호 생성, 되돌리기 어려움

### 11.2 묶음 처리

- **목적** — 동일 받는사람 주문 합포장
- **활성화** — 체크박스 2건 이상 선택 시 [묶음] 버튼 활성
- **절차**
  1. 대상 주문 체크박스 선택 (2건 이상)
  2. [묶음] 클릭
  3. 기준 주문 확인
  4. 묶음 대상 주문 목록 확인
  5. [묶음 확정] 클릭
- ⚠️ 처리 후 개별 주문 합산, 결제금액·상품 구성 확인 필수

## 13. 원산지 유무 설정

### 13.1 개별 설정

- 상품 상세 테이블 원산지(유무) 열에서 [있음] / [없음] 선택

### 13.2 일괄 설정

- [원산지 유무 설정] 버튼 → 전체 일괄 설정 모달

- [전체 없음] — 모든 상품 원산지 없음으로 설정

## 14. 공통 기능

### 14.1 바코드 출력

- 상품 행 바코드 아이콘 클릭 → 미리보기 모달
- 출력 매수 입력 (50~500개)
- [다운로드] — 이미지 파일 다운로드
- [직접출력] — 즉시 프린터 출력

- [관리자 메모 > 등록/보기] → 모달에서 내용 입력·저장
