## 1. 화면 개요

- **페이지명** — 매입 관리
- **목적** — 서비스별 결제 매입 현황 조회 및 집계
- **304 정산매출/매입 관리와의 차이**
  - 304: 매출·매입을 함께 조회하며 마진·수수료 수익 분석 포함
  - 305: 매입(구매 원가) 중심으로 조회, 추가비 항목 포함
- **주요 수행 업무**
  - 기간별 매입 현황 조회
  - 상품값·구매비·현지배송비·추가비 합계 집계
  - 쿠폰 할인 적용 내역 확인
  - 결제일자 기준 정렬 및 검색

## 2. 상단 집계 통계 패널

화면 상단에 현재 조회 기간의 매입 집계 수치를 표시한다.

| 패널 항목 | 설명 |
|----------|------|
| 청구금액 합계 | 전체 청구금액 합계 |
| 결제금액 합계 | 실제 결제된 금액 합계 |
| 예치금 합계 | 예치금 차감 합계 |
| 쿠폰 할인 합계 | 쿠폰 적용 할인 합계 |
| 상품값 합계 | CNY 기준 상품값 합계 |
| 구매비 합계 | 원화 기준 구매비 합계 |
| 현지배송비 합계 | 현지 배송비 합계 |
| 추가비 합계 | 추가 비용 합계 |

## 3. 검색 및 필터

### 3.1 패널 열기·닫기

- [상세 검색] 헤더 클릭 → 검색 폼 펼침 / 접힘

### 3.2 기간

| 필드 | 설명 |
|------|------|
| 기간 설정 | 시작일 ~ 종료일 / [오늘] 버튼 |

### 3.3 서비스 유형 및 결제 관련 필터

| 필드 | 선택 옵션 |
|------|----------|
| 서비스유형 | 전체 / 배송비용 / 구매비용 / 수입비용 / 시장조사 / 반품비 / 창고비 |
| 결제타입 | 관리자 예치금결제 / 예치금 결제 / 무통장입금 / 카드 |

### 3.4 번호 검색

| 필드 | 설명 |
|------|------|
| 사서함번호 | 고객 사서함번호 |
| 이름 | 회원명 |
| 주문번호 | 주문 고유번호 |
| 청구번호 | 청구 고유번호 |
| 결제번호 | 결제 고유번호 |

### 3.5 실행 및 초기화

- [조회] — 조건으로 목록 필터링
- [초기화] — 전체 조건 기본값 복원
