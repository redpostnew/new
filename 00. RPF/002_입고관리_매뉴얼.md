## 1. 화면 개요

- **페이지명** — 입고 관리
- **목적** — 물류센터 입고 내역을 확인하고 상태를 관리
- **주요 수행 업무**
  - 입고 예정 화물 조회 및 현황 모니터링
  - 바코드·QR 스캔으로 입고 처리
  - 노데이터(미매칭 화물) 등록 및 고객 매핑
  - 입고 상태 변경 (입고예정·부분입고·입고완료)
  - 검수 이동 및 출고 요청

---

## 2. 입고 업무 흐름

### 2.1 기본 흐름

```
입고예정
    ↓
바코드/QR 스캔 or 수동 조회
    ↓
주문 정보 자동 매칭
    ↓
입고 완료 → 상태 자동 업데이트
```

### 2.2 매칭 실패 시 (노데이터) 흐름

```
노데이터 스캔
    ↓
임시 보관 처리
    ↓
관리자 수동 사서함 매칭 → 주문 연결
    ↓
상품 상태 확인
    ├─ 수량 부족 → 부분 수량 확정 → 추가 입고 대기
    ├─ 오류입고 → 고객 대기 (고객사 대기 상태)
    └─ 처리 완료 → 주문 상태 업데이트
```

### 2.3 상태 코드

| 상태 | 설명 |
|------|------|
| 입고예정 | 입고 예정 화물 등록 상태 |
| 부분입고 | 주문 수량 일부만 도착한 상태 |
| 입고완료 | 전체 수량 입고 완료 |
| 고객확인대기중 | 상품 이상으로 고객 확인 대기 중 |

## 6. 바코드 스캔 입고 처리

### 6.1 스캔 입고 프로세스 개요

카메라로 송장 바코드를 스캔하여 입고를 자동 처리한다.

### 6.2 스캔 방법

- **카메라 스캔** — 바코드를 프레임 안에 맞추면 트래킹번호 자동 인식
- **수기 조회** — 스캔 불가 시 번호를 직접 입력 후 조회

### 6.3 스캔 결과 처리

| 결과 | 처리 방법 |
|------|----------|
| 정상 스캔 (주문서 매칭 성공) | 상품 조회 성공 → 입고 완료 자동 처리 |
| 노데이터 스캔 (주문서 미매칭) | 노데이터 등록 화면으로 이동 → 수동 매핑 |

### 6.4 QR 코드 스캔

- 카메라로 QR 코드를 인식하거나 코드 직접 입력
- 스캔 중 상태 표시 → 스캔 결과 표시

## 7. 노데이터(미매칭) 관리

트래킹번호로 주문서를 찾을 수 없는 경우 노데이터로 처리한다.

### 7.1 노데이터 등록 화면

스캔 결과 화면에서 [노데이터 등록] 진행

| 항목 | 설명 |
|------|------|
| 주문 선택 (선택사항) | 아래 목록에서 해당 주문 선택 |
| 사서함 | 고객 사서함번호 검색 (선택사항) |
| 신청방법 | 접수 방법 선택 |
| 사서함 번호 | 사서함 번호 직접 입력 |
| 사진 첨부 | 화물 사진 드래그 또는 클릭 업로드 |

> 📌 검수 진행은 입고 즉시 시작됩니다. 주문 상태와 무관하게 진행됩니다.

### 7.2 노데이터 관리 탭

- 수동 등록된 노데이터 목록 표시
- 고객 사서함 검색 → 노데이터 화물과 주문 연결
- 노데이터 항목 없음 메시지 표시 시 모든 화물 정상 매칭 완료

## 8. 입고 상태 변경

### 8.1 일괄 상태 변경

1. 변경할 주문 체크박스 선택
2. [상태변경] 드롭다운에서 변경 상태 선택
3. [변경] 클릭 → 확인 팝업에서 확인

### 8.2 변경 가능 상태

| 상태 | 설명 |
|------|------|
| 접수신청 | 신청 접수 완료 처리 |
| 부분입고 | 일부 수량만 입고 처리 |
| 입고완료 | 전체 입고 완료 처리 |
| 검수완료 | 검수 완료 처리 (검수 이동 없이 직접 완료) |
| 신청취소 | 신청 취소 처리 |
| 고객확인대기중 | 상품 이상으로 고객 확인 대기 상태 전환 |

### 8.3 상품별 업무처리 선택

행을 펼쳐 상품 상세 테이블에서 개별 상품의 처리 방법 선택 가능

- **검수이동** — 입고 후 즉시 검수 진행
- **출고요청** — 검수 없이 바로 출고 요청

> ⚠️ 상태 변경은 되돌리기 어려우므로 반드시 확인 후 진행

## 9. 검수 이동 · 출고 요청

### 9.1 검수 이동

- **목적** — 입고 완료 상품을 검수 관리 페이지로 이동
- **절차**
  1. 검수 이동할 주문 체크박스 선택
  2. [검수이동] 버튼 클릭
  3. 검수 관리 페이지에서 해당 건 조회 가능

### 9.2 출고 요청

- **목적** — 검수 없이 직접 출고 요청
- **절차**
  1. 출고 요청 대상 주문 체크박스 선택
  2. [출고요청] 버튼 클릭
  3. 출고 관리 페이지에서 해당 건 조회 가능

## 10. 주문 유형별 처리

### 10.1 배송대행 입고

- 고객이 직접 중국에서 발송한 상품 입고 처리
- 트래킹번호로 주문 자동 매칭
- 매칭 실패 시 사서함번호로 수동 연결

### 10.2 구매대행 입고

- 담당자가 1688 등에서 구매한 상품 입고 처리
- 1688주문번호와 연결된 송장번호로 자동 매칭
- 상품 수량·옵션 일치 여부 확인 필수

### 10.3 수입대행 입고

- 수입 화물 입고 처리
- 통관 후 물류센터 도착 시 처리
- 영문상품명(통관품목) 정확히 입력되어 있어야 통관 정보 매칭 가능

## 11. 공통 기능

### 11.1 바코드 출력

- 상품 상세 테이블 바코드(유/무) 출력 아이콘 클릭 → 미리보기 모달
- 출력 매수 입력 (50~500개)
- [다운로드] 또는 [직접출력]

### 11.2 관리자 메모

- [관리자 메모 > 등록/보기] 클릭 → 모달에서 내용 입력·저장
- 저장 시마다 이력 자동 누적, 고객 비노출

### 11.3 물류센터 요청사항

- [물류센터 요청사항 > 상세보기] 클릭 → 요청사항 입력·저장
- 최신 내용만 저장 (이력 없음)

### 11.4 부가서비스 등록

- 상품 행 [부가서비스] 등록 버튼 클릭 → 설정 모달
- 서비스 선택 → 수량·비용 입력 → 관리자 확인 완료 체크 → 저장

### 11.5 추가기능 메뉴

| 메뉴 | 설명 |
|------|------|
| 주문 관리 | 주문 관리 페이지로 이동 |
| 바코드 출력 | 바코드 출력 |
| 부가서비스 상세/확인 | 부가서비스 확인 |
