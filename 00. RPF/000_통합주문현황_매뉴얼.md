## 1. 화면 개요

- **페이지명** — 통합 주문현황
- **목적** — 구매대행 · 배송대행 · 수입대행 전체 주문 현황을 하나의 화면에서 확인하고 관리
- **주요 수행 업무**
  - 서비스 유형별 · 단계별 주문 건수 모니터링
  - 전체 주문 목록 조회 · 검색 · 필터링
  - 주문 상태 일괄 변경
  - 나눔 · 묶음 처리
  - 구매견적 · 수입견적 산출 및 청구
  - 고객사서함 정보 · 주문서 상세 조회

## 2. 전체 업무 흐름 구조

통합 주문현황 화면은 주문부터 배송완료까지 모든 단계의 상태를 한 화면에서 조회한다.

### 2.1 서비스 유형별 상태 흐름

**구매대행**
```
구매견적 → 결제대기 → 결제완료 → 구매완료
→ 입고완료 → 부분입고 → 고객확인 대기중
→ 검수중 → 오류입고 → 오류처리중 → 검수완료
→ 재고전환신청 → 재고등록
→ 출고요청 → 포장완료 → 포장오류 → 쿠팡서류 대기 → 출고보류 → 출고완료 → 배송완료
```

**수입대행**
```
수입견적 → 결제대기 → 결제완료 → 구매완료
→ (이후 구매대행과 동일 흐름)
```

**배송대행**
```
접수대기 → 접수신청 → 고객확인 대기중
→ (이후 입고·검수·출고 흐름)
```

### 2.2 부가 업무 상태

| 업무 | 상태 코드 |
|------|----------|
| 반품관리 | 반품중 → 환불 확인 중 |
| 시장조사 | 조사중 → 조사완료 |

### 2.3 전체 상태 코드 일람

| 단계 | 상태 |
|------|------|
| 주문 | 구매견적 · 수입견적 · 결제대기 · 결제완료 · 구매완료 |
| 배송대행 | 접수대기 · 접수신청 |
| 입고 | 입고예정 · 부분입고 · 입고완료 · 고객확인 대기중 |
| 검수 | 검수중 · 오류입고 · 오류처리중 · 검수완료 |
| 재고 | 재고전환신청 · 재고등록 |
| 출고 | 출고요청 · 포장완료 · 포장오류 · 쿠팡서류 대기 · 출고보류 · 출고완료 · 배송완료 |
| 공통 | 신청취소 · 반품중 · 환불 확인 중 |

## 3. 상태 현황 패널

화면 상단에 서비스 유형별·단계별 주문 건수를 카드 형태로 표시한다.

### 3.1 패널 구성

| 패널 | 표시 상태 |
|------|----------|
| 구매대행 | 구매견적 / 결제대기 / 결제완료 / 구매완료 |
| 수입대행 | 수입견적 / 결제대기 / 결제완료 / 구매완료 |
| 배송대행 | 접수대기 / 접수신청 |

### 3.2 카운트 클릭 필터

- 각 상태의 건수 클릭 → 해당 유형·상태로 목록 자동 필터링
- 별도 검색 없이 즉시 해당 건 확인 가능

## 6. 상태 일괄 변경

### 6.1 사용 방법

1. 변경할 주문 행의 체크박스 선택
2. [상태변경] 드롭다운에서 변경할 상태 선택
3. [변경] 버튼 클릭
4. 확인 팝업에서 건수·상태 확인 후 [변경하기] 클릭

### 6.2 변경 가능 상태

| 상태 | 설명 |
|------|------|
| 신청취소 | 주문 신청 취소 |
| 입고완료 | 물류센터 입고 완료 처리 |
| 부분입고 | 일부 수량만 입고된 상태 |
| 검수완료 | 검수 완료 처리 |
| 포장완료 | 포장 완료 처리 |
| 출고완료 | 출고 완료 처리 |

> ⚠️ 상태 변경은 되돌리기 어려우므로 반드시 선택 건수와 변경 상태를 확인 후 진행

### 6.3 주문 상태 변경 모달

- 현재 상태와 변경 가능 상태를 나란히 표시
- 원하는 상태 선택 후 [변경하기] 클릭

## 7. 나눔(Split) · 묶음(Merge)

### 7.1 나눔 처리

- **목적** — 상품을 사이즈·종류별로 나눠 배송
- **활성화 조건** — 체크박스 2건 이상 선택
- **절차**
  1. 나눔 대상 주문 체크박스 선택 (2건 이상)
  2. [나눔] 버튼 클릭
  3. 나눔 모달에서 현재 주문정보 확인
  4. 나눌 제품 목록에서 상품·수량 설정
  5. [나눔 확정] 클릭
- **나눔 처리 내역** — 나눔수량 / 잔여수량 표시, 바코드 출력 가능
- ⚠️ 처리 후 원래 주문 분리 → 새 주문번호 생성, 되돌리기 어려움

### 7.2 묶음 처리

- **목적** — 동일 받는사람 주문을 합포장
- **활성화 조건** — 체크박스 2건 이상 선택
- **절차**
  1. 묶음 대상 주문 체크박스 선택 (2건 이상)
  2. [묶음] 버튼 클릭
  3. 묶음 모달에서 기준 주문 확인
  4. 묶음 대상 주문 목록 확인 (이미지·상품명·상품옵션 표시)
  5. [묶음 확정] 클릭
- ⚠️ 처리 후 개별 주문이 합산됨 → 결제금액·상품 구성 확인 필수

## 10. 구매견적 · 수입견적 모달

### 10.1 구매견적 (구매대행)

**원가 정보 입력**

| 항목 | 설명 | 입력 방식 |
|------|------|----------|
| 구매금액 (CNY) | 현지 상품 금액 | 자동 표시 |
| 현지 배송비 (CNY) | 중국 내 배송비 | 자동 표시 |
| 위안환율 (원/CNY) | CNY → KRW 환율 | 직접 입력 |
| 달러환율 (원/USD) | USD → KRW 환율 | 직접 입력 |

**수수료 및 세금 (자동 계산)**

| 항목 | 설명 |
|------|------|
| 회원구매 수수료 | 구매대행 수수료율 (기본 10%, 수정 가능) |
| 구매금액 (원) | 환율·수수료 적용 후 금액 |
| 부가세 | 구매금액의 10% |
| 총 청구금액 | 구매금액(원) + 부가세 |

**자동 계산 공식**
```
구매금액(원) = 구매금액(CNY) × 위안환율 × (1 + 수수료율/100)
부가세       = 구매금액(원) × 0.1
총 청구금액  = 구매금액(원) + 부가세
```

- [청구하기] 클릭 → 결제대기 자동 전환 + 카톡 알림 발송

### 10.2 수입견적 (수입대행)

**총비용 내역 (자동 계산)**

| 항목 | 설명 |
|------|------|
| 상품가 | 현지 상품 금액 |
| 중국 배송비 | 중국 내 배송비 |
| 수수료 (레포 수수료) | 대행 수수료 |
| 국제 운송비 | 국제 운송 비용 |
| 관세 (상품별) | 품목별 관세 |
| 통관 수수료 | 통관 처리 비용 |
| 국내운송비 | 국내 배송비 |
| 마진율 | 모든 비용 선계산 후 25~30% 적용 |
| 공급가 (원) | 최종 공급가 |
| 부가세 | 전체비용의 10% |

- 실제 마진율 자동 계산 표시
- [청구하기] 클릭 → 결제대기 전환 + 카톡 알림 발송

## 11. 공통 기능

### 11.1 바코드 출력

- 상품 상세 테이블의 바코드 아이콘 클릭 → 미리보기 모달
- 출력 매수 입력 (50~500개)
- [다운로드] 또는 [직접출력] 선택

### 11.2 운송장 출력

- 한국운송장 열의 프린터 아이콘 클릭 → 운송장 라벨 인쇄

### 11.3 관리자 메모

- [등록/보기] 클릭 → 관리자 메모 모달
- 내용 입력 후 저장 → 저장 일시·내용 이력 자동 누적
- 고객 비노출, 이력 삭제 불가

### 11.4 물류센터 요청사항

- [상세보기] 클릭 → 요청사항 모달
- 최신 내용만 저장 (이력 관리 없음)

### 11.5 상품 메모

- 상품 행 [등록/보기] 클릭 → 상품 메모 모달
- 메모 내용 입력·저장, 이력 조회 가능

### 11.6 부가서비스

- 상품 행 [부가서비스] 등록 버튼 클릭 → 부가서비스 설정 모달
- 서비스 선택 → 수량·비용 입력 → 관리자 확인 완료 체크 → 저장
- **관리자 확인 완료** — 최종 검수 및 부가서비스 내용 확정

### 11.7 첨부파일

| 아이콘 | 유형 | 활성 색상 |
|--------|------|----------|
| 이미지 아이콘 | 사진·이미지 | 파란색 (미등록: 회색) |
| 문서 아이콘 | PDF·Word 등 | 초록색 (미등록: 회색) |
| 파일 아이콘 | 기타 파일 | 주황색 (미등록: 회색) |
