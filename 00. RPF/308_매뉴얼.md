## 1. 화면 개요

- **페이지명** — 정산통계
- **목적** — 검색기간 비용 종합 분석
- **주요 수행 업무**
  - 월별 서비스 유형별 매출·순이익·주문건수 집계
  - 구매대행·배송대행·수입대행·부가서비스 통합 분석
  - 이익률 산출 및 총 매출 통계 확인
  - 결제방법별(예치금·카드·현금) 현황 분석

## 2. 검색 및 필터

### 2.1 기간 설정

| 필드 | 설명 |
|------|------|
| 시작 월 | 통계 시작 월 선택 (월 단위) |
| 종료 월 | 통계 종료 월 선택 (월 단위) |

- 날짜 단위가 아닌 **월(Month) 단위**로 기간 설정
- 기간 범위가 길수록 추세 분석에 유리

### 2.2 서비스 유형 필터

| 필드 | 선택 옵션 |
|------|----------|
| 서비스유형 | 전체 / 구매대행 / 배송대행 / 수입대행 |

### 2.3 결제방법 필터

| 필드 | 선택 옵션 |
|------|----------|
| 결제방법 | 전체 / 예치금 / 카드 / 현금 |

### 2.4 실행 및 초기화

- 필터 설정 후 자동 조회 또는 [조회] 버튼 클릭
- [초기화] — 전체 조건 기본값 복원

## 3. 통계 집계 패널

### 3.1 서비스별 합계

화면 상단에 조회 기간의 서비스별 통계 합계를 표시한다.

| 항목 | 설명 |
|------|------|
| 총 구매대행 | 기간 내 구매대행 총 매출액 |
| 총 배송대행 | 기간 내 배송대행 총 매출액 |
| 총 수입대행 | 기간 내 수입대행 총 매출액 |
| 총 부가서비스 | 기간 내 부가서비스 총 매출액 |
| 총 매출 | 전체 서비스 통합 총 매출액 |

### 3.2 성과 지표

| 지표 | 설명 |
|------|------|
| 이익률 | (순이익 / 매출액) × 100 (%) |
| 매출액 | 기간 내 전체 매출액 |
| 주문건수 | 기간 내 전체 주문 처리 건수 |
| 순이익 | 매출액 - 매입 원가 = 순이익 |

## 5. 월별 통계 활용 방법

### 5.1 추세 분석

- 시작 월 ~ 종료 월을 넓게 설정하면 월별 추세 분석 가능
- 전월 대비 증감률로 성장세 파악
- 비수기·성수기 패턴 확인 가능

### 5.2 서비스 유형별 비교

- 구매대행·배송대행·수입대행 이익률 비교
- 이익률이 높은 서비스에 집중 운영 전략 수립
- 부가서비스 비중 확인으로 추가 수익원 파악

### 5.3 결제방법별 분석

- 예치금·카드·현금 결제 비율 확인
- 예치금 활성화로 운전자금 흐름 최적화 가능

### 5.4 주요 지표 해석 기준

| 지표 | 기준 | 설명 |
|------|------|------|
| 이익률 | 높을수록 양호 | 매출 대비 순이익 비율 |
| 주문건수 | 증가 추세 파악 | 물량 성장 지표 |
| 순이익 | 플러스 유지 | 원가 대비 수익성 지표 |
| 총 매출 | 월별 성장률 확인 | 전체 사업 규모 지표 |
