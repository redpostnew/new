## 1. 화면 개요

- **페이지명** — 예치금
- **목적** — 회원 예치금 사용 및 입금 현황 통합 관리
- **주요 수행 업무**
  - 예치금 보유 현황 조회 (KRW / CNY 통화별)
  - 예치금 적립 내역 관리 (고객 입금 → 예치금 전환)
  - 예치금 사용 내역 조회 (결제 시 차감 이력)
  - 예치금 환급 처리 (미사용 예치금 고객 반환)
  - 관리자 예치금 직접 결제 상세 조회

## 2. 예치금 업무 개요

### 2.1 예치금 종류

| 구분 | 통화 | 설명 |
|------|------|------|
| KRW 예치금 | 원화 | 원화 기준 예치금 |
| CNY 예치금 | 위안화 | 위안화 기준 예치금 |

### 2.2 예치금 흐름

```
고객 입금 (무통장/이체)
    ↓
예치금 적립 (KRW 또는 CNY)
    ↓
결제 시 자동 차감 (예치금 사용)
    ↓
잔액 보유 또는 환급 신청
    ↓
예치금 환급 처리
```

### 2.3 예치금 구분 코드

| 구분 | 설명 |
|------|------|
| 예치금 적립 | 고객 입금으로 예치금 충전 |
| 예치금 사용 | 결제 시 예치금 차감 |
| 예치금 환급 | 보유 예치금 고객 반환 |

### 2.4 업무처리 상태

| 상태 | 설명 |
|------|------|
| 신청 | 환급·적립 신청 완료, 처리 전 |
| 완료 | 처리 완료 |
| 취소 | 처리 취소됨 |

## 3. 상단 집계 패널

화면 상단에 현재 조회 기간의 예치금 집계 수치를 통화별로 표시한다.

| 패널 항목 | 설명 |
|----------|------|
| 보유예치금 | 현재 보유 예치금 총액 (KRW / CNY) |
| 예치금 적립 | 기간 내 적립된 예치금 합계 |
| 예치금 사용 | 기간 내 사용된 예치금 합계 |
| 예치금 환급 | 기간 내 환급된 예치금 합계 |

각 패널 항목 클릭 시 해당 구분으로 목록 자동 필터링

## 6. 예치금 적립 처리

### 6.1 개요

- 고객이 무통장·이체로 입금한 금액을 예치금으로 등록하는 업무
- KRW 또는 CNY 통화 선택하여 적립 가능

### 6.2 적립 처리 절차

1. [예치금 적립] 탭 또는 버튼 클릭
2. 해당 고객 입금 건 확인
3. 입금자명·금액·통화·내용/계좌 확인
4. 처리 상태 → **완료** 전환

### 6.3 주요 확인 항목

| 항목 | 설명 |
|------|------|
| 통화 | KRW / CNY 확인 |
| 금액 | 입금 금액 정확 여부 확인 |
| 입금자명 | 입금자 이름이 회원명과 일치 여부 확인 |
| 내용 / 계좌 | 입금 계좌 및 내용 메모 |

---

## 7. 예치금 사용 처리

### 7.1 개요

- 고객이 결제 시 예치금을 차감하여 사용한 내역을 조회
- 관리자 예치금 직접 결제의 경우 별도 처리 가능

### 7.2 예치금 사용 내역 조회

- [예치금 사용] 탭 → 기간별 사용 내역 표시
- 주문번호·결제번호와 연결된 사용 이력 확인 가능

### 7.3 관리자 예치금 결제 처리

- 관리자가 고객 예치금으로 직접 결제 처리
- [관리자 예치금 결제 상세] 팝업에서 주문번호 조회 후 처리 (9장 참고)

---

## 8. 예치금 환급 처리

### 8.1 개요

- 고객이 보유한 잔여 예치금을 반환 처리

### 8.2 환급 처리 절차

1. [예치금 환급] 탭 → 환급 신청 건 조회
2. 환급 신청 상태의 건 확인
3. 환급 금액·계좌 정보 확인
4. 실제 이체 처리 후 상태 → **완료** 전환

### 8.3 주요 확인 항목

| 항목 | 설명 |
|------|------|
| 통화 | KRW / CNY 구분 확인 |
| 금액 | 환급 신청 금액 확인 |
| 내용 / 계좌 | 환급 계좌 정보 확인 |
| 업무처리 | 완료 처리 전 신청 상태 확인 |

---

## 9. 관리자 예치금 결제 상세

### 9.1 개요

- 관리자가 고객 예치금을 직접 차감하여 결제 처리할 때 사용
- 달러 송금·실환율·적용환율 등 환전 정보 포함

### 9.2 모달 구성

| 항목 | 설명 |
|------|------|
| 주문번호 조회 | 처리할 주문번호 조회 |
| 결제 금액 | 결제 처리 금액 (원화) |
| 송금 달러 | 달러 기준 송금 금액 |
| 실제 입금 달러 | 실제 입금된 달러 금액 |
| 실환율 | 실제 송금 시 적용 환율 |
| 적용환율 | 시스템 적용 환율 |

### 9.3 처리 절차

1. 해당 예치금 사용 행에서 [관리자 예치금 결제 상세] 클릭
2. 주문번호 조회로 결제 대상 주문 확인
3. 결제 금액·환율 정보 입력
4. [확인] 클릭 → 예치금 차감 및 결제 처리 완료
