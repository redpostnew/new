## 1. 화면 개요

- **페이지명** — 정산매출/매입 관리
- **목적** — 서비스별 결제 정산매출/매입 현황 조회 및 집계
- **주요 수행 업무**
  - 기간별 정산매출·매입 현황 조회
  - 청구금액·결제금액·예치금·마진·수수료 수익 합계 집계
  - 상품값(위안)·적용환율·구매비·수수료·현지배송비·부가세 내역 확인
  - 부가서비스 상세 내역 조회

## 2. 상단 집계 통계 패널

화면 상단에 현재 조회 기간의 주요 집계 수치를 표시한다.

| 패널 항목 | 설명 |
|----------|------|
| 청구금액 합계 | 전체 청구금액 합계 (KRW N건 / CNY N건) |
| 예치금 사용 합계 | 예치금 사용 N건 합계 |
| 구매비(상품가) 합계 | 전체 N건 기준 구매비 합계 |
| 위안화 상품값 합계 | CNY 기준 상품값 합계 |
| 부가세 합계 | 부가세 발생 N건 합계 |
| 상품값(위안) 합계 | KRW N건 / CNY N건 기준 |
| 평균 실송금환율 | 기간 내 평균 실송금 환율 |
| 구매 수수료 합계 | 구매 수수료 전체 합계 |
| 현지 배송비 합계 | 현지 배송비 전체 합계 |

추가 집계 항목 (하단)

| 패널 항목 | 설명 |
|----------|------|
| 결제금액 합계 | 실제 결제된 금액 합계 |
| 예치금 합계 | 예치금 차감 합계 |
| 마진 합계 | 매출-매입 마진 합계 |
| 수수료 수익 합계 | 대행 수수료 수익 합계 |

## 3. 검색 및 필터

### 3.1 패널 열기·닫기

- [상세 검색] 헤더 클릭 → 검색 폼 펼침 / 접힘

### 3.2 기간

| 필드 | 설명 |
|------|------|
| 기간 설정 | 시작일 ~ 종료일 / [오늘] 버튼 |

### 3.3 정산 구분

| 필드 | 선택 옵션 |
|------|----------|
| 정산매출/매입 | 전체 / 정산매출 / 정산매입 |

- **정산매출** — 고객으로부터 받은 매출 기준 조회
- **정산매입** — 현지 구매 등 매입 기준 조회

### 3.4 서비스 유형 및 결제 관련 필터

| 필드 | 선택 옵션 |
|------|----------|
| 서비스유형 | 전체 / 배송비용 / 구매비용 / 수입비용 / 시장조사 / 반품비 / 창고비 |
| 결제타입 | 관리자 예치금결제 / 예치금 결제 / 무통장입금 / 카드 |
| 결제화폐 | KRW / CNY |

### 3.5 번호 검색

| 필드 | 설명 |
|------|------|
| 사서함번호 | 고객 사서함번호 |
| 이름 | 회원명 |
| 주문번호 | 주문 고유번호 |
| 청구번호 | 청구 고유번호 |
| 결제번호 | 결제 고유번호 |

### 3.6 실행 및 초기화

- [조회] — 조건으로 목록 필터링
- [초기화] — 전체 조건 기본값 복원

## 4. 정산 목록 테이블

### 4.1 테이블 컬럼 구조

| 컬럼 | 설명 |
|------|------|
| 결제일시 | 결제 처리 일시 |
| 결제번호 | 결제 고유번호 |
| 회원명/ID | 회원명과 로그인 ID |
| 사서함번호 | 고객 사서함번호 |
| 결제구분 | 결제 서비스 유형 |
| 결제타입 | 결제 수단 유형 |
| 청구번호 | 연결된 청구번호 |
| 결제화폐 | KRW / CNY 구분 |
| 주문번호 | 연결된 주문번호 |
| 청구금액 | 청구된 원래 금액 |
| 결제금액 | 실제 결제 금액 |
| 예치금 | 예치금 차감 금액 |
| 상품값(위안) | CNY 기준 상품값 |
| 적용환율 | 결제 시 적용된 환율 |
| 수수료% | 적용된 수수료율 |
| 구매비(상품가) | 구매 상품가 (원) |
| 구매 수수료 | 구매 대행 수수료 |
| 현지배송비 | 현지 배송비 (원) |
| 현지 배송비(위안) | 현지 배송비 (CNY) |
| 부가세 | 부가세 금액 |
| 부가서비스 | 부가서비스 비용 |
| 실송금환율 | 실제 송금 시 적용 환율 |

### 4.2 컬럼 주요 계산 관계

```
구매비(상품가) = 상품값(위안) × 적용환율 × (1 + 수수료%/100)
구매 수수료   = 구매비(상품가) × 수수료%/100
부가세        = 구매비(상품가) × 부가세율
마진          = 청구금액 - 구매비(상품가) - 현지배송비 - 부가세 - 부가서비스
```
