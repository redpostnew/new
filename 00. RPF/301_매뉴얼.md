## 1. 화면 개요

- **페이지명** — 청구관리 (묶음청구 / 청구하기)
- **목적** — 서비스 유형별 청구번호 생성 및 고객 청구 처리
- **주요 수행 업무**
  - 출고 완료 주문에 대한 청구번호 생성 (묶음 또는 개별)
  - 출고·물류비 산출 및 고객 청구 처리
  - 창고비 별도 청구
  - 청구 취소 및 상태 복구

## 2. 청구 업무 흐름

### 2.1 기본 흐름

```
포장완료 / 쿠팡서류대기 상태의 주문
    ↓
청구번호 생성 (묶음 또는 개별)
    ↓
부가서비스·물류비 입력 및 확인
    ↓
청구하기 → 화주에게 결제 요청 알림 발송
    ↓
청구완료
```

### 2.2 청구 취소 시

```
청구 취소
    ↓
해당 주문서 → 포장완료 / 쿠팡서류대기 상태로 복구
(부가서비스·중량 데이터는 유지)
```

### 2.3 청구 구분

| 구분 | 설명 |
|------|------|
| 묶음청구 | 여러 주문을 1개의 청구번호로 묶어 청구 |
| 개별청구 | 각 주문을 개별 청구번호로 생성하여 청구 |

### 2.4 청구 상태 코드

| 상태 | 설명 |
|------|------|
| 청구번호 유 | 청구번호가 생성된 주문 |
| 청구번호 무 | 아직 청구번호 미생성 주문 |
| 청구완료 | 청구가 완료된 주문 |
| 취소됨 | 청구가 취소된 주문 |

  > `선택된 주문에 [서비스유형]이 섞여 있습니다. 서비스유형이 같은 주문끼리만 묶음 청구가 가능합니다.`

## 5. 청구번호 생성 (묶음)

### 5.1 개요

- 여러 주문을 1개의 청구번호로 결합
- 동일 서비스유형 주문끼리만 묶음 청구 가능

### 5.2 절차

1. 묶음 청구할 주문 체크박스 선택 (2건 이상)
2. [청구번호 생성 (묶음)] 버튼 클릭
3. 묶음 청구 생성 모달 열림

  | 항목 | 설명 |
  |------|------|
  | 선택 건수 | 선택된 주문 수량 |
  | 생성 청구번호 | 자동 생성될 청구번호 미리보기 |
  | 청구번호 접두어 (선택) | 청구번호 앞에 붙일 접두어 직접 입력 가능 |
  | 메모 | 청구 관련 메모 입력 |

4. 내용 확인 후 [청구번호 생성] 클릭
5. 선택 건이 1개의 청구번호로 결합됨

### 5.3 엮인 주문번호 확인

- 묶음 청구 생성 후 청구번호에 엮인 주문번호 목록 확인 가능
- 청구 방식(묶음/개별) 표시

## 6. 청구번호 생성 (개별)

### 6.1 개요

- 선택한 각 주문을 독립된 청구번호로 각각 생성
- 서비스유형 혼재 주문 선택 시 자동으로 개별 청구 적용

### 6.2 절차

1. 개별 청구할 주문 체크박스 선택
2. [청구번호 생성 (개별)] 버튼 클릭
3. 개별 청구 생성 모달 열림

  | 항목 | 설명 |
  |------|------|
  | 선택 건수 | 선택된 주문 수량 |
  | 생성 청구번호 | N건 → N개의 청구번호 생성 안내 |

4. [청구번호 생성] 클릭
5. 각 주문에 독립된 청구번호 부여

## 7. 청구하기 처리

### 7.1 개요

- 청구번호가 생성된 주문에 대해 실제 청구 처리
- 청구 완료 시 화주(고객)에게 결제 요청 알림 발송

### 7.2 청구하기 모달 구성

청구번호 선택 후 [청구하기] 클릭 → 청구하기 모달 열림

**기본 정보**

| 항목 | 설명 |
|------|------|
| 발행된 청구번호 | 현재 청구번호 |
| 사서함 번호 | 고객 사서함번호 |
| 회원명 | 청구 대상 회원 |
| 출고거래처 선택 | 거래처A / 거래처B / 거래처C / 거래처D 중 선택 |
| 서비스 | 서비스 유형 |
| 물류방식 | 운송 방법 |
| 주소지 | 배송지 주소 |
| 예치금 자동차감 신청여부 | 신청 / 신청안함 선택 |

**상품 정보**

| 항목 | 설명 |
|------|------|
| 상품번호 | 상품 고유 번호 |
| 이미지 | 상품 이미지 |
| 상품명 | 상품명 |
| 옵션 | 상품 옵션 |
| 단가 | 단가 (원) |
| 수량 | 수량 |
| 소계 | 소계 (원) |

**부가서비스 항목**

| 항목 | 설명 |
|------|------|
| 파렛트 적재 | 팔레트 적재 작업비 |
| 원산지증명서 CO | 원산지 증명서 발행 비용 |
| 송장 출력 및 부착 | 송장 출력·부착 작업비 |
| 핸들링차지 DO | 핸들링차지 비용 |
| 쿠팡 서류 부착 | 쿠팡 서류 부착 작업비 |
| 컨테이너 작업비 | 컨테이너 작업 비용 |
| 관리자 부가서비스 추가 | 관리자가 직접 부가서비스 항목 추가 가능 |

**비용 합계**

| 항목 | 설명 |
|------|------|
| 부가서비스 합계 | 등록된 부가서비스 총합 |
| 물류비용 | 출고·물류 비용 |
| 관리자 추가 부가서비스 비용 | 관리자 추가 항목 비용 |
| 부피/무게 | 박스 수량 / 박스 중량 정보 |
| 총 청구금액 | 최종 청구 금액 |

### 7.3 청구 완료

- 내용 확인 후 [청구 완료] 버튼 클릭
- 화주(고객)에게 결제 요청 알림 자동 발송
- 주문 상태 → **청구완료** 전환

## 8. 창고비 청구

### 8.1 개요

- 재고 보관 상품의 창고 보관비를 별도 청구하는 기능
- 서비스유형 선택 시 **창고료** 항목 선택

### 8.2 창고비 청구 모달 구성

| 항목 | 설명 |
|------|------|
| 박스사이즈 | 소(S) / 중(M) / 대(L) / 특대(XL) 선택 |
| 박스 수량 | 창고 보관 박스 수량 입력 |
| 상품명 | 보관 상품명 |
| 상품코드 | 보관 상품코드 |
| 색상 | 색상 정보 |
| 사이즈 | 사이즈 정보 |
| 수량 | 보관 수량 |
| 위치 | 창고 내 랙 위치 |
| 보관일자 | 보관 시작 일자 |
| 비용 | 청구 비용 입력 |
| 메모 | 창고비 관련 메모 |

### 8.3 창고비 청구 절차

1. [창고비 청구] 버튼 클릭
2. 창고비 청구 모달에서 정보 입력
3. 내용 확인 후 [청구하기] 클릭
4. 고객에게 창고비 결제 요청 발송

## 9. 청구 취소

### 9.1 개요

- 발행된 청구번호를 취소하고 주문을 이전 상태로 복구

### 9.2 취소 후 상태 복구

- 해당 주문서 → **포장완료** 또는 **쿠팡서류대기** 상태로 복구
- 부가서비스·중량 데이터는 유지

### 9.3 청구 취소 절차

1. 취소할 청구 주문 행 선택
2. [청구취소] 버튼 클릭 또는 [추가기능 ▾] → [청구취소] 선택
3. 청구 취소 확인 팝업

  | 항목 | 설명 |
  |------|------|
  | 취소 확인 메시지 | 취소 후 상태 안내 |
  | [청구 취소] | 취소 확정 버튼 |
  | [아니오] | 취소 없이 닫기 |

4. [청구 취소] 클릭 → 청구 취소 처리 완료

> ⚠️ 청구 취소는 되돌릴 수 없으므로 반드시 확인 후 진행
