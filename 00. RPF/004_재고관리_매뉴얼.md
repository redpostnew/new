## 1. 화면 개요

- **페이지명** — 재고 관리
- **목적** — 재고 현황을 모니터링하고 입출고 정보를 관리
- **주요 수행 업무**
  - 재고 전환 신청 상품의 랙 위치 지정 및 재고 등록
  - 재고 현황 조회 (경과일·창고비·가용수량 확인)
  - 재고 출고 요청 처리
  - 업무처리 상태 관리 (미처리·처리중·처리완료·보류)
  - 원산지 유무 설정
  - 바코드 출력·부가서비스 등록

## 2. 재고 업무 흐름

### 2.1 기본 흐름

```
일반 구매대행·배송대행 주문에서 재고 전환 신청
    ↓
재고전환신청 (재고(보관) 선택 후 신청)
    ↓
재고등록 — 관리자가 랙번호 부여 후 적재
    ↓
재고(보관) 기간 운영 (경과일·창고비 발생)
    ↓
재고출고요청
    ↓
재고출고 태그 부여 → 출고 관리로 이동
```

### 2.2 재고 전환 경로

- **일반 경로** — 구매대행·배송대행 주문에서 재고 보관 선택 후 신청
- **재고 미거치 경로** — 재고를 거치지 않고 바로 출고로 이동하는 상품

### 2.3 상태 코드

| 상태 | 설명 |
|------|------|
| 재고전환신청 | 재고 보관 신청 완료, 등록 대기 중 |
| 재고등록 | 랙 위치 지정 및 재고 DB 등록 완료 |
| 재고출고 | 재고에서 출고 처리된 상태 (태그 부여) |

## 6. 재고 등록

### 6.1 재고 등록 모달 열기

- **방법 1** — [상태변경] 드롭다운에서 [재고등록] 선택 후 [변경] 클릭
- **방법 2** — [추가기능 ▾] → [재고등록] 선택

### 6.2 재고 등록 모달 구성

**주문 확인**

| 항목 | 설명 |
|------|------|
| 주문번호 | 등록할 주문번호 확인 |
| 상품 확인 | 등록 대상 상품 정보 확인 |

**재고 위치 지정**

| 항목 | 설명 |
|------|------|
| 재고 위치 | `구역-선반-위치` 형식으로 입력 |
| 입력 예시 | A-3-12 (A구역 3번 선반 12번 위치) |

**박스 정보 입력**

| 항목 | 설명 |
|------|------|
| 가로 (cm) | 박스 가로 치수 |
| 세로 (cm) | 박스 세로 치수 |
| 높이 (cm) | 박스 높이 치수 |
| 삼변합 (cm) | 가로+세로+높이 자동 계산 |
| 박스 수량 | 박스 개수 |
| 입수량 | 박스당 상품 입수 수량 |

### 6.3 재고 등록 완료 처리

1. 위치·박스 정보 입력 완료
2. [저장] 클릭
3. 주문 상태 → **재고등록** 전환
4. 재고 목록에 랙 위치·재고입고일시 표시

## 7. 재고 상태 변경

### 7.1 일괄 상태 변경

1. 대상 주문 체크박스 선택
2. [상태변경] 드롭다운에서 상태 선택
3. [변경] 클릭 → 확인 팝업에서 확인

### 7.2 변경 가능 상태

| 상태 | 설명 |
|------|------|
| 재고등록 | 랙 위치 지정 및 재고 DB 등록 |
| 출고요청 | 재고에서 출고 요청 처리 |
| 신청취소 | 재고 신청 취소 |
| 반품신청 | 반품 처리 신청 |

> ⚠️ 상태 변경은 되돌리기 어려우므로 반드시 확인 후 진행

## 8. 재고 출고 요청

### 8.1 재고출고요청 모달

재고에서 출고를 요청할 때 사용한다.

| 항목 | 설명 |
|------|------|
| 출고 요청 주문번호 확인 | 출고 요청할 주문번호 표시 |
| 재고출고요청 주문 | 요청 대상 주문 목록 |
| 선택된 배송지 | 출고 배송지 확인 |

### 8.2 출고 요청 절차

1. 출고 요청 대상 주문 체크박스 선택
2. [상태변경] → [출고요청] 선택 후 [변경] 클릭
3. 재고출고요청 모달에서 주문번호·배송지 확인
4. [확인] 클릭 → 출고 관리 페이지로 이동

### 8.3 경과일 및 창고비 확인

- **경과일** — 재고 입고 후 현재까지 경과한 일수
- **창고비** — 경과일에 따라 자동 산정된 창고 보관 비용
- 출고 요청 전 창고비를 반드시 확인하여 고객에게 안내

## 9. 업무처리 상태 관리

### 9.1 상태 종류

| 상태 | 색상 | 의미 |
|------|------|------|
| 미처리 | 회색 | 아직 처리하지 않은 상태 (기본값) |
| 처리중 | 노란색 | 현재 처리 진행 중 |
| 처리완료 | 초록색 | 처리 완료 |
| 보류 | 보라색 | 처리 보류 중 |

### 9.2 업무처리 상태 변경 방법

1. 해당 주문 행의 [업무 상태 처리] 버튼 클릭 또는 업무처리 드롭다운 선택
2. 업무 상태 처리 모달 열림

  | 항목 | 설명 |
  |------|------|
  | 현재 상태 | 현재 업무처리 상태 표시 |
  | 상태 변경 | 변경할 상태 선택 |
  | 처리 메모 | 처리 내용 메모 입력 |

3. 상태 선택 → 처리 메모 입력 → [저장] 클릭

### 9.3 재고 이력 조회

- 각 주문의 재고 처리 이력 조회 가능
- 이력 없는 경우 `조회된 데이터가 없습니다.` 표시

## 10. 원산지 유무 설정

### 10.1 개별 설정

- 상품 상세 테이블 원산지 열에서 [있음] / [없음] 선택

### 10.2 일괄 설정

- [원산지 유무 설정] 버튼 클릭 → 전체 일괄 설정 모달
- [전체 있음] — 모든 상품 원산지 있음으로 설정
- [전체 없음] — 모든 상품 원산지 없음으로 설정

## 11. 공통 기능

### 11.1 바코드 출력

- 상품 행 바코드 아이콘 클릭 → 미리보기 모달
- 출력 매수 입력 (50~500개) → [다운로드] / [직접출력]

### 11.2 관리자 메모

- [관리자 메모 > 등록/보기] → 모달에서 내용 입력·저장
- 저장 시마다 이력 자동 누적, 고객 비노출

### 11.3 물류센터 요청사항

- [물류센터 요청사항 > 상세보기] → 요청사항 입력·저장
- 최신 내용만 저장 (이력 없음)

### 11.4 부가서비스 등록

- 상품 행 [부가서비스] 등록 버튼 → 설정 모달
- 서비스 선택 → 수량·비용 입력 → 관리자 확인 완료 체크 → [저장]

### 11.5 첨부파일 관리

| 아이콘 | 유형 | 활성 색상 |
|--------|------|----------|
| 이미지 | 사진·이미지 | 파란색 |
| 문서 | PDF·Word 등 | 초록색 |
| 파일 | 기타 파일 | 주황색 |

### 11.6 추가기능 메뉴

| 메뉴 | 설명 |
|------|------|
| 주문서 상세 | 주문 상세 정보 조회 |
| 주문서 수정 | 주문서 내용 수정 |
| 바코드 출력 | 바코드 출력 |
| 입금요청 | 고객 입금 요청 발송 |
| 견적취소 | 발행된 견적 취소 |
| 부가서비스 상세/확인 | 등록된 부가서비스 확인 |
| 원산지 유무 설정 | 원산지 표기 여부 설정 |
