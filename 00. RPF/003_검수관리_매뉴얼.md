## 1. 화면 개요

- **페이지명** — 검수 관리
- **목적** — 물류센터 검수 내역을 확인하고 상태를 관리
- **주요 수행 업무**
  - 입고 완료 상품에 대한 실사 검수 처리
  - 실사 이미지 등록 및 고객 전달
  - 원산지 확인 (있음/없음)
  - 오류 상품 처리 (반품·그냥수령·선택적 처리)
  - 패킹 처리 (박스 구성 및 번호 지정)
  - 상품 QR 출력 및 부가서비스 확인

## 2. 검수 업무 흐름

### 2.1 정상 검수 흐름

```
입고완료
    ↓
검수상품 스캔 (트래킹스캔)
    ↓
실사 촬영 → 고객 확인 전달
    ↓
부가서비스 확인·처리
    ↓
검수완료
    ↓
출고요청 이동 또는 재고입고
```

### 2.2 오류 발생 시 흐름

```
오류 발생 (주문서 불일치·판매불가·피드백 대기)
    ↓
┌─────────────────────────────────┐
│  그냥수령  │  선택적  │  반품    │
│ 문제 상품  │ 일부만   │ 판매자에 │
│ 정상 진행  │ 처리     │ 반송     │
└─────────────────────────────────┘
```

### 2.3 검수 후 분기

- **출고요청** — 부가서비스 출고요청 시 출고 관리 이동
- **재고입고** — 체크 시 출고 대신 재고관리로 이동 (재고 보관 선택 시)

### 2.4 상태 코드

| 상태 | 설명 |
|------|------|
| 고객확인대기중 | 실사 촬영 후 고객 확인 대기 중 |
| 검수중 | 검수 처리 진행 중 |
| 오류입고 | 상품 오류로 입고된 상태 |
| 오류처리중 | 오류 상품 처리 진행 중 |
| 검수완료 | 검수 모든 과정 완료 |

## 6. 검수 처리 절차

### 6.1 운송장 스캔으로 검수 시작

1. 운송장 바코드를 스캐너로 스캔 또는 번호 직접 입력 후 엔터
2. 스캔 성공 → 해당 주문의 상품 목록 표시
3. 스캔 실패 → `해당 운송장번호를 찾을 수 없습니다` 메시지 → 수기 조회

### 6.2 검수 처리 모달 구성

| 항목 | 설명 |
|------|------|
| 검수 상품 | 검수 대상 상품 목록 |
| 실사 사진 | 실사 이미지 업로드 |
| 고객답변확인/처리 | 고객 확인 결과에 따른 처리 선택 |
| 부가서비스 선택 | 해당 주문 부가서비스 확인 및 선택 |
| 상태별 수량 | 정상 수량 / 오류 수량 / 합계 수량 확인 |
| 처리 결과 선택 | 검수 처리 결과 선택 (정상·오류·반품 등) |

### 6.3 검수 상태 처리

| 항목 | 설명 |
|------|------|
| 현재 마스터 상태 | 주문의 현재 검수 상태 |
| 주문수량 | 전체 주문 수량 |
| 마스터 상태 변경 | 검수 결과에 따라 상태 변경 |
| 정상 수량 | 이상 없는 수량 |
| 오류 수량 | 문제 발생 수량 |
| 합계 수량 확인 | 정상 + 오류 = 전체 수량 일치 여부 |

> ⚠️ 검수처리 비활성 — `검수중` 상태에서만 처리 가능

### 6.4 실사 사진 업로드

1. 상품 상세 테이블 실사사진 영역에서 [사진 선택 / 카메라 촬영] 클릭
2. 최대 6장까지 업로드 가능
3. 사진 업로드 완료 후 고객 전달 가능 (고객확인대기중 전환)
4. ⚠️ 실사 사진을 먼저 업로드해야 고객답변확인/처리 가능

### 6.5 고객 확인 처리

실사 촬영 완료 후 고객에게 전달 → 고객 답변에 따라 처리 방향 선택

| 고객 답변 | 처리 방향 |
|----------|----------|
| 정상 수령 | 정상 프로세스 진행 |
| 이상 있음 | 오류 처리 진행 |

### 6.6 재고 처리 선택

검수 처리 시 **재고입고** 체크박스 선택 시 → 출고 대신 재고관리로 이동

```
재고입고 체크 → 재고관리 페이지에 재고로 등록
재고입고 미체크 → 정상 출고 프로세스 진행
```

### 6.7 검수 처리 소요시간

- 작업 완료 체크리스트 확인 후 소요시간 자동 기록
  - 상품 바코드 박스 내부 부착 완료
  - 박스 외부 번호 처리 완료 (수기 기재 또는 바코드 부착)
  - 수량 확인 완료

## 7. 오류 처리

### 7.1 오류 유형

| 오류 상태 | 설명 |
|----------|------|
| 오류입고 | 주문서 불일치·판매불가·피드백 대기 등으로 오류 등록 |
| 오류처리중 | 오류 확인 후 처리 방법 결정 중 |
| 고객확인대기중 | 고객에게 실사 사진 전달 후 확인 대기 |

### 7.2 오류 처리 방법

| 처리 방법 | 설명 |
|----------|------|
| 그냥수령 | 문제 상품 그냥 수령하여 정상 프로세스 진행 |
| 선택적 | 일부 상품만 선택하여 처리 |
| 반품 | 판매자에게 상품 반송 |
| 오류무시 | 오류를 무시하고 진행 |

### 7.3 오류 처리 절차

1. 오류 입고 상품 확인
2. 실사 사진 촬영 및 업로드
3. 고객에게 실사 사진 전달 (고객확인대기중 전환)
4. 고객 답변 확인
5. 처리 방법 선택 (그냥수령·선택적·반품)
6. [완료] 클릭 → 오류처리 완료

## 8. 원산지 유무 설정

### 8.1 개별 설정

- 상품 상세 테이블 원산지(유무) 열에서 [있음] / [없음] 선택

### 8.2 일괄 설정

- [원산지 유무 설정] 버튼 → 전체 일괄 설정 모달
- [전체 있음] — 모든 상품 원산지 있음으로 설정
- [전체 없음] — 모든 상품 원산지 없음으로 설정

## 9. 패킹 처리 (박스 구성)

### 9.1 패킹 모달 구성

| 항목 | 설명 |
|------|------|
| 총 박스 수 | 전체 박스 개수 입력 |
| 현재 박스 번호 | 현재 작업 중인 박스 번호 |
| 박스 정보 | 박스별 상세 정보 입력 |
| 사업자명 | 사업자 이름 |
| 영문사업자명 | 영문 사업자 이름 |
| 도착지 | 화물 도착지 |
| 담당자 연락처 | 담당자 연락처 |

### 9.2 박스 치수 입력

| 항목 | 설명 |
|------|------|
| 가로 (cm) | 박스 가로 치수 |
| 세로 (cm) | 박스 세로 치수 |
| 높이 (cm) | 박스 높이 치수 |
| 삼변합 (cm) | 가로+세로+높이 자동 계산 |
| 박스 수량 | 박스 내 상품 수량 |
| 입수량 | 박스 당 입수 수량 |

### 9.3 재고 위치 입력

- `구역-선반-위치` 형식으로 입력
- 예: `A-3-12` (A구역 3번 선반 12번 위치)

### 9.4 박스번호 QR 출력

- 패킹 모달에서 [박스번호 QR 이미지] 출력 버튼
- 박스 내부 부착용 QR과 상품 QR 구분 가능하도록 출력
- [인쇄 미리보기] → [인쇄하기] 클릭

### 9.5 작업 완료 체크리스트

패킹 완료 전 다음 항목 확인

- ☐ 상품 바코드 박스 내부 부착 완료
- ☐ 박스 외부 번호 처리 완료 (수기 기재 또는 바코드 부착)
- ☐ 수량 확인 완료

## 10. 상품 QR 출력 및 스캔

### 10.1 상품 QR 출력

1. 상품 상세 테이블 상품QR출력 열 [QR출력] 버튼 클릭
2. QR 코드 출력 모달 열림
3. 출력 부수 입력
4. [인쇄하기] 클릭

### 10.2 상품 QR 스캔 확인

- 카메라로 상품 QR 스캔 → 상품과 일치 여부 확인
