## 1. 화면 개요

- **페이지명** — 부가서비스 검수/승인
- **목적** — 부가서비스를 검수하고 승인/반려 처리
- **주요 수행 업무**
  - 작업자가 완료한 부가서비스 작업 내역 조회
  - 작업 이미지·수량·불량 수량 확인
  - 관리자 승인(확인) 또는 반려 처리
  - 정산비 확인 및 부가서비스 비교

---

## 2. 부가서비스 업무 흐름

### 2.1 작업 흐름

```
부가서비스 등록 (주문 관리·검수 관리에서 등록)
    ↓
작업자 배정 및 작업 시작 (진행중)
    ↓
작업 완료 (작업수량·불량수량·작업시간 기록)
    ↓
관리자 검수 (이미지·수량 확인)
    ↓
┌──────────────────────────┐
│  확인(승인)  │  반려      │
│  처리 완료   │  재작업    │
└──────────────────────────┘
```

### 2.2 작업 상태 코드

| 상태 | 설명 |
|------|------|
| 대기 | 작업 배정 전 대기 상태 |
| 진행중 | 작업자 작업 진행 중 |
| 완료 | 작업 완료, 관리자 확인 대기 |
| 반려 | 관리자 반려, 재작업 필요 |

### 2.3 부가서비스 종류

| 종류 | 설명 |
|------|------|
| 원산지작업 | 원산지 라벨 부착·확인 작업 |
| 포장 | 특수 포장 작업 |
| 검수 | 상품 검수 작업 |
| 불량검수 | 불량 여부 검수 작업 |
| 시간당 알바 | 시간당 단위 알바 작업 |
| 박스갈이 | 기존 박스를 새 박스로 교체 |

## 3. 검색 및 필터

### 3.1 패널 열기·닫기

- [상세 검색] 헤더 클릭 → 검색 폼 펼침 / 접힘

### 3.2 기간

| 필드 | 설명 |
|------|------|
| 작업날짜 | 작업 처리 날짜 시작일 ~ 종료일 / [오늘] 버튼 |

### 3.3 주문·상품 검색

| 필드 | 설명 |
|------|------|
| 주문번호 | 주문 고유번호 |
| 상품명 | 한글 상품명 |

### 3.4 회원 검색

| 필드 | 설명 |
|------|------|
| 회원명 | 고객 회원명 |
| 회원 ID | 고객 로그인 ID |
| 사서함번호 | 고객 사서함번호 |

### 3.5 작업자 검색

| 필드 | 설명 |
|------|------|
| 작업자이름 | 작업 담당자 이름 |
| 작업자 ID | 작업 담당자 ID |

### 3.6 상태 및 종류 필터

| 필드 | 선택 옵션 |
|------|----------|
| 작업상태 | 전체 / 대기 / 진행중 / 완료 / 반려 |
| 부가서비스 종류 | 전체 / 원산지작업 / 포장 / 검수 / 불량검수 / 시간당 알바 / 박스갈이 |

### 3.7 실행 및 초기화

- [조회] — 조건으로 목록 필터링
- [초기화] — 전체 조건 기본값 복원

## 4. 작업 목록 테이블

### 4.1 상단 카운터

- **총 N건** — 현재 조회 건수 실시간 표시

### 4.2 테이블 컬럼 구조

| 컬럼 | 설명 |
|------|------|
| (체크박스) | 행 선택 |
| 상품이미지 | 작업 대상 상품 이미지 |
| 작업/불량 이미지 | 작업 완료 후 촬영된 이미지 |
| 주문번호 | 연결된 주문번호 |
| 회원명/회원ID/사서함번호 | 고객 정보 통합 표시 |
| 작업자이름 | 작업 담당자 이름 |
| 옵션명(한글) | 상품 한글 옵션명 |
| 작업상태 | 현재 작업 상태 배지 |
| 주문수량 | 총 주문 수량 |
| 총완료수량 | 전체 완료된 수량 |
| 작업수량 | 이번 작업에서 완료된 수량 |
| 불량수량 | 불량으로 처리된 수량 |
| 남은수량 | 아직 미완료된 잔여 수량 |
| 고객 청구비용 | 고객에게 청구되는 비용 |
| 작업시간 | 작업 소요 시간 |
| 상품번호 | 상품 고유번호 |
| 상품명 | 한글 상품명 |
| 작업자ID | 작업자 로그인 ID |
| 옵션(중문) | 중국어 옵션명 |
| 작업현황 | 작업 진행 현황 요약 |
| 부가서비스 비교 | 요청 부가서비스와 작업 내용 비교 |
| 알바 정산비 | 시간당 알바 정산 비용 |
| 관리자 확인/반려 | 승인·반려 처리 버튼 영역 |

### 4.3 수량 관계

```
주문수량 = 작업수량 + 불량수량 + 남은수량
총완료수량 = 작업수량 + 불량수량
```

## 5. 부가서비스 상세/확인 모달

### 5.1 모달 열기

- 목록 테이블 행 클릭 또는 [부가서비스 상세/확인] 버튼 클릭

### 5.2 모달 구성

**이미지 미리보기**

| 항목 | 설명 |
|------|------|
| 작업 이미지 | 작업 완료 후 촬영한 이미지 표시 |
| 등록된 이미지가 없습니다 | 이미지 미등록 시 표시 |

**작업한 부가서비스 내역**

| 항목 | 설명 |
|------|------|
| 부가서비스 항목 | 작업 완료된 부가서비스 목록 |
| 수량 합계 | 전체 수량 합계 표시 |

**수량 합계 요약**

| 항목 | 설명 |
|------|------|
| 고객요청 | 고객이 요청한 작업 수량 |
| 작업완료 | 실제 완료된 작업 수량 |
| 미해당 | 해당 없는 항목 수량 |
| 미처리 | 미처리된 항목 수량 |

**관리자 확인 완료 처리**

- [관리자 확인 완료] 버튼 — 최종 검수 및 부가서비스 내용 확정
- 확정 후에는 수정 불가

## 6. 확인 (승인) 처리

### 6.1 개요

- 작업 완료된 부가서비스를 관리자가 검토 후 확인(승인) 처리
- 확인 완료 시 정산·청구 단계로 진행 가능

### 6.2 확인 처리 절차

1. 완료 상태의 작업 행에서 [확인] 버튼 클릭
2. 부가서비스 상세/확인 모달 열림
3. 작업 이미지 확인 — 작업 내용이 요청과 일치하는지 검토
4. 수량 확인 — 작업수량·불량수량·남은수량 검토
5. 부가서비스 비교 — 요청 항목과 완료 항목 일치 여부 확인
6. [관리자 확인 완료] 클릭 → 최종 승인 처리
7. 작업 상태 → **완료** (확인) 전환

### 6.3 확인 전 체크 항목

| 체크 항목 | 설명 |
|----------|------|
| 이미지 등록 여부 | 작업 이미지가 첨부되어 있는지 확인 |
| 작업수량 일치 | 주문수량과 완료수량 일치 여부 |
| 불량수량 처리 | 불량 수량에 대한 처리 방향 확인 |
| 부가서비스 일치 | 요청 부가서비스 항목과 작업 내용 일치 |
| 정산비 적절성 | 청구비용·알바 정산비 적절성 검토 |

## 7. 반려 처리

### 7.1 개요

- 작업 내용이 기준에 미달하거나 이미지·수량 불일치 시 반려 처리
- 반려 시 작업자에게 재작업 요청

### 7.2 반려 처리 절차

1. 완료 상태의 작업 행에서 [반려] 버튼 클릭
2. 부가서비스 상세/확인 모달에서 반려 사유 확인
3. 반려 사유 입력 (작업 내용 불일치, 이미지 미등록 등)
4. [반려] 클릭 → 작업 상태 → **반려** 전환
5. 작업자에게 반려 알림 발송

### 7.3 반려 주요 사유

| 반려 사유 | 설명 |
|----------|------|
| 이미지 미등록 | 작업 완료 이미지 미첨부 |
| 수량 불일치 | 주문수량과 완료수량 불일치 |
| 작업 품질 미달 | 작업 결과가 기준 이하 |
| 요청 항목 누락 | 요청된 부가서비스 항목 일부 누락 |

## 8. 이미지 미리보기

### 8.1 이미지 확인

- 부가서비스 상세/확인 모달의 [이미지 미리보기] 섹션
- 작업자가 업로드한 작업 완료 이미지 표시
- 이미지 미등록 시: `등록된 이미지가 없습니다.` 표시

### 8.2 이미지 확인 기준

| 확인 항목 | 기준 |
|----------|------|
| 이미지 등록 여부 | 최소 1장 이상 등록 |
| 이미지 품질 | 작업 결과 식별 가능 수준 |
| 작업 내용 확인 | 요청한 부가서비스 내용 이미지에서 확인 가능 |
